# AI Unit Test Generator

Leverages the power of artificial intelligence to automate the generation of unit tests.

## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/blake-nx/AI-Unit-Test-Generator.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
